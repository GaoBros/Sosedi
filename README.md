# Веб-приложение "Соседи"

## Перед запуском

1. Запустить XAMPP (рекомендуется версия 3.3.0 или выше)  
2. Включить Apache  
3. Включить MySQL  
4. Скачать файл `sosedi.sql`  
5. Открыть phpMyAdmin  
6. Импортировать скачанный файл

## Для запуска 
1. Запустить XAMPP (рекомендуется версия 3.3.0 или выше)  
2. Включить Apache  
3. Включить MySQL
4. Открыть в браузере ссылку http://localhost/sosedi/index.php
